# Design System & Under Construction Pages — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Build a Vercel/Geist-inspired design system with 4 themed under-construction pages featuring unique GSAP SVG animations, plus a design system showcase page.

**Architecture:** CSS custom properties for design tokens, GSAP for SVG animations, three-state theme toggle (system/light/dark) with localStorage persistence. Asymmetric split layout — text left, animation right. TanStack Start routes with FSD file organization.

**Tech Stack:** React 19, TanStack Start/Router, GSAP + @gsap/react, Geist fonts, Tailwind CSS v4, CSS custom properties.
